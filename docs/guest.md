# ゲストの管理

ゲスト一覧ページでは、席次表に表示するゲストの追加・編集・削除や、CSV を使った一括登録・更新ができます。
席次表へのゲストの配席方法については[席次表の使い方](seating-chart.md)をご覧ください。

## 1. ゲストを追加する {#add-guest}

1. ゲスト一覧ページで「ゲストを追加」ボタンをタップします
2. 以下の情報を入力します
   - **名前**(必須)
   - **敬称**(例:様、さん、くん)
   - **肩書き**(例:新郎友人、新婦同僚)
   - **メールアドレス**
3. 「追加」ボタンをタップして保存します

## 2. ゲストを CSV で一括登録・更新する {#import-guests-csv}

CSV ファイルから複数のゲストをまとめて登録(新規)、または既存のゲストを一括で更新できます。
事前にまとめておかれたゲストリストの取り込みや、登録済みゲストの一斉修正にご利用いただけます。

### 2-1. CSV のフォーマット {#import-guests-csv-format}

- **ヘッダー行**:1 行目は `ゲストID,ゲスト名,敬称,肩書き` の順に固定です
- **行数**:最大 500 件(ヘッダー行を除く)
- **文字コード**:UTF-8 / UTF-8 (BOM 付き) / Shift_JIS のいずれにも対応
- **改行コード**:LF / CRLF / CR のいずれにも対応

### 2-2. ゲストID 列の役割 {#import-guests-csv-id-column}

`ゲストID` 列の値によって、その行が新規登録か更新かが切り替わります。

| `ゲストID` の値 | 動作 |
|---|---|
| 空欄 | **新規登録**(INSERT) |
| 既存ゲストの ID | その既存ゲストを **更新**(UPDATE) |

ゲストIDは「現在のゲスト一覧 CSV」をダウンロードしたときに自動的に
出力されます。手動で考えて入力する必要はありません。

### 2-3. 新規一括登録の手順 {#import-guests-csv-insert}

1. ゲスト一覧ページで「インポート」ボタンをタップします
2. モーダル内の「テンプレート CSV をダウンロード」リンクからテンプレートを取得します
3. テンプレートを編集して、登録したいゲストの情報を入力します(`ゲストID` 列は空のまま)
4. 「ファイルを選択」から作成した CSV ファイルを選択します
5. 「インポート」ボタンをタップします
6. 成功すると登録件数のメッセージが表示され、ゲスト一覧が更新されます

### 2-4. 既存ゲストの一括更新の手順 {#import-guests-csv-update}

1. ゲスト一覧ページで「エクスポート」ボタンをタップして、現在のゲスト一覧 CSV を
   ダウンロードします
   - 「インポート」モーダル内の「現在のゲスト一覧 CSV をダウンロード」リンクからも
     同じ CSV を取得できます
2. ダウンロードした CSV を編集して、ゲスト名・敬称・肩書きを変更します
   - `ゲストID` 列は **変更しないでください**(変更すると別のゲストの更新と
     見なされるか、ID が見つからずエラーになります)
3. 「インポート」ボタンをタップしてモーダルを開きます
4. 「ファイルを選択」から編集後の CSV を選択します
5. 「インポート」ボタンをタップします
6. 該当行の既存ゲストが更新されます

!!! warning "編集時の注意"
    Excel で CSV を開くと、ゲストIDが「数値」と判定されて表記が崩れることがあります。
    可能であればテキストエディタ(VS Code など)での編集をおすすめします。
    Excel で編集される場合は、ゲストID 列を「文字列」として読み込むよう
    ご注意ください。

!!! warning "エラーが発生したとき"
    取り込めなかった行はモーダル内に **行番号・列名・原因** とともに一覧表示されます。
    エラーが 1 件でもあるとインポートは**全件中止**されますので、
    表示されたエラーをすべて修正してから再度アップロードしてください。

    主なエラー例:

    - `ゲストID が見つかりません`:該当 ID のゲストが存在しない
    - `ゲストID が N 行目と重複しています`:同じ ID が CSV 内に複数行ある
    - `ゲスト名 は必須です`:名前列が空欄
    - `ゲスト名 は255文字以内で入力してください`:文字数超過

## 3. ゲストを編集する {#edit-guest}

1. ゲスト一覧で編集したいゲストの「編集」ボタンをタップします
2. 名前・敬称・肩書き・メールアドレスを変更します
3. 「保存」ボタンをタップします
4. 確認ダイアログが表示されるので「更新」をタップします

!!! warning "ご注意"
    ゲスト情報の変更は公開済みの席次表にも即時反映されます。
    誤字脱字の修正や敬称の変更は問題ありませんが、まったく別の人物への変更はおすすめしません。

## 5. よくある質問 {#faq}

### Q. ゲスト情報を変更すると公開済みの席次表にも反映されますか? {#faq-edit-reflect}

はい。ゲスト一覧でゲスト情報を変更すると、公開済みの席次表にも即時反映されます。

### Q. ゲストを削除するとどうなりますか? {#faq-delete-effect}

ゲストを削除すると、席次表上のそのゲストの席は空席になります。公開済みの席次表にも即座に反映されるのでご注意ください。
公開済みの席次表に存在しないゲストを削除しても、席次表には影響ありません。

### Q. CSV インポートで一部の行だけエラーになった場合、エラーのない行は登録されますか? {#faq-import-csv-partial}

いいえ、エラーが 1 件でもあるとインポートは**全件中止**されます。エラー行も正常な行も含めて 1 件も登録されないため、エラーメッセージに従ってすべての行を修正してから再度アップロードしてください。

### Q. 取り込めるゲストの上限はありますか? {#faq-import-csv-limit}

1 度のインポートで最大 500 件までです。これを超える場合はファイルを分割してアップロードしてください。

### Q. エクスポートとテンプレートの違いはなんですか? {#faq-export-vs-template}

- **テンプレート**:ゲストがまだ登録されていないとき、CSV のフォーマットの参考用にサンプル行 1 件入りのファイルが手に入ります
- **エクスポート**:既に登録済みのゲストを CSV で書き出します。一括更新の起点として使ってください

「インポート」モーダル内のリンクは、ゲストが 0 件のときは「テンプレート CSV をダウンロード」、1 件以上のときは「現在のゲスト一覧 CSV をダウンロード」と挙動が自動で切り替わります。

### Q. ゲストID は手動で入力できますか? {#faq-guest-id-manual}

ゲストIDは ULID 形式の固定値で、システムが自動で発行します。手動で考案したり別のシステムから持ち込んだりすることはできません。
新規登録時は **空欄のまま** にしてください。
